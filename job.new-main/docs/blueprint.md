# **App Name**: CareerCompass

## Core Features:

- Job Opportunity Aggregator: Fetch and display job postings from LinkedIn and Internshala based on user-defined career options and location.  Styled job cards are used with hover effects, color, and shadows.
- AI Interview Simulator: Interview Simulator providing practice interview sessions, leveraging generative AI to ask questions that are contextually relevant based on the user's stated goals. The LLM will act as a tool when deciding what information to incorporate into its questioning strategy.
- Roadmap Generator: Personalized roadmap creation tool to guide users through the necessary steps for their desired career path.
- Gamified XP Sidebar: Gamified experience point (XP) sidebar showing user progress and badge achievements.
- Live News Feed: Display live, categorized news cards related to tech and job market trends.  Each is decorated with relevant emojis to create an engaging visual element.
- Theme Toggle: Toggle to switch between light and dark UI themes.
- Animated Background and Transitions: Subtle SVG background animations and transitions.
- Advanced Job Filtering: Allow users to filter job listings by platform (LinkedIn, Internshala), location, and minimum salary.
- Badge Display: Showcase badges earned by users for completing profile setup, applying to jobs, completing mock interviews, mastering resume building, and achieving application milestones.
- Interview Question Tracker: Allow users to track previously asked interview questions and categorize them by difficulty (easy, medium, hard).
- Coding Challenge Module: Integrate coding challenges and problems inspired by LeetCode, allowing users to practice and improve their coding skills.
- Resume Analyzer: AI-powered resume analyzer that provides feedback and suggestions for improvement.
- Calendar Integration: Calendar integration for scheduling interviews and tracking application deadlines.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for calmness and trust.
- Background color: Light gray (#F0F8FF) for a clean and professional look.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern tech look, Body font: 'Inter' (sans-serif).
- Consistent and modern icon set, using flat design for UI elements, and brand-relevant icons for news cards.
- Clean and structured layout with intuitive navigation, search filters and easily accessible sections.
- Use subtle transition effects for page navigation and interactive elements (buttons, cards).
